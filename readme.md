# How to Get a Free Gemini API Key

**What is an API key?** Think of it like a password that lets your app or project talk to Google's Gemini AI. You'll need one to use Gemini in your own projects or tools.

The whole process takes about **5 minutes** and is completely free.

---

## What You'll Need

- A Google account (the same one you use for Gmail or YouTube)
- A web browser (Chrome, Firefox, Safari — whatever you use)

That's it. No credit card, no special software.

---

## Step 1 — Open Google AI Studio

Open your browser and go to this address:

**aistudio.google.com**

This is Google's official website for developers to access Gemini AI.

---

## Step 2 — Sign In

Click **"Get Started"** in the top-right corner of the page.

Sign in with your Google account the same way you would for Gmail or YouTube. If you're already signed into Google, it may do this automatically.

---

## Step 3 — Accept Google's Terms of Service

A pop-up will appear asking you to agree to Google's rules for using their AI tools.

Read through it if you like, then click **"I agree"** to continue. There's also an optional checkbox to receive emails from Google about AI updates — that part is up to you.

---

## Step 4 — Go to the API Keys Page

Once you're in, look at the **left sidebar** (the menu on the left side of the screen).

Click on **"Get API key"**. This takes you to the page where you can create and manage your keys.

> You can also go directly to: **aistudio.google.com/apikey**

---

## Step 5 — Create Your Key

On the API Keys page, click the button that says **"Create API Key"**.

A small window will pop up. It will ask which project to attach the key to.

- If you're new, just click **"Create API key in new project"** — this is the easiest option and sets everything up for you automatically.
- If you've used Google Cloud before, you can choose an existing project.

Then click **"Create"** and your key will be generated in a few seconds.

---

Your API key will appear on screen. It looks something like this:

**This is important:** Copy it right now and save it somewhere safe before closing the window. Good places to store it:

- A notes app on your phone or computer (something private, not a shared document)
- A password manager if you use one
- A plain text file saved somewhere only you can access

> ⚠️ **Never share your API key publicly.** Don't post it in a forum, paste it into a chat, or upload it to GitHub. If someone else gets your key, they can use Gemini AI on your account.

---

## Step 7 — Use Your Key

When you're ready to use the key in a project or tool, you'll typically paste it into a settings field or a configuration file. The exact spot depends on what you're building or using.

The key is usually stored in something called an **environment variable** — which is just a way of keeping the key separate from your main code so it stays private. If you're following a tutorial or using a specific tool, it will tell you exactly where to put it.

---

## What's Free and What Isn't

The free tier gives you access to Gemini's most capable everyday models, including:

- **Gemini 2.0 Flash** — fast, smart, great for most tasks
- **Gemini 2.5 Flash** — Google's latest efficient model

This is more than enough for learning, experimenting, and small personal projects.

Some of Google's most advanced models (like Gemini 3.1 Pro) require a paid plan. But you won't need those to get started.

> **Note on privacy:** On the free tier, Google may use your prompts and responses to improve their AI. If that's a concern, you can upgrade to a paid plan later.

---

## Something Went Wrong?

| Problem | What to do |
|---|---|
| Page says "not available in your country" | The free tier isn't available everywhere. Try enabling billing on your project (you won't be charged unless you go over the limits). |
| Key doesn't seem to work | Wait a minute or two after creating it — sometimes it takes a moment to activate. |
| You hit a usage limit | You've made too many requests in a short time. Wait a bit and try again, or switch to a lighter model. |
| You lost your key | Go back to aistudio.google.com/apikey, delete the old one, and create a new one. |

---

## You're Done!

You now have a free Gemini API key. From here, you can follow any Gemini tutorial or tool setup guide using this key wherever it asks for one.

If you ever need to manage, rename, or delete your keys, just head back to **aistudio.google.com/apikey** anytime.
